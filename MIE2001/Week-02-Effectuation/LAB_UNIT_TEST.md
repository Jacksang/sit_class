# Week 2 — Lab & Unit Test Design

## Lab: "My Effectual Profile" (50 min)

### Objective
Apply all 5 effectuation principles to your own life. By the end, you'll have a profile you can use to evaluate any venture idea this semester.

### Part A: Personal Effectual Profile (20 min, individual)
Complete the worksheet (from PPT outline). For each principle, write concrete answers — not generic ones.

**Key pointers:**
- **Bird-in-Hand:** Be specific. Not "I know some people" but "My uncle runs a logistics company in JB. My friend is good at TikTok. I learned Python in Year 1."
- **Affordable Loss:** Be honest. Not "I can invest \$10,000" (probably not true for a student) but "I can spend \$100 and 5 hours per week."
- **Crazy Quilt:** Think beyond founders. Who on campus could help? (SIT I&E office, professors, alumni, classmates, family friends)
- **Lemonade:** Pick a real setback from the past 6 months. If you can't find opportunity in it, why not? Is there a principle you're not applying?
- **Pilot-in-the-Plane:** This should be the simplest principle. Action > analysis. What's ONE thing?

### Part B: Case Analysis — Effectuation in a Real Startup (15 min, groups)
**Find a partner or group of 3.** Pick ONE Singapore startup (or startup-like initiative). Identify which effectuation principles they used.

**You can choose:**
- An example from the PPT (Carousell, Doctor Anywhere, Grab)
- A company you know personally
- A food stall / small business you visit regularly

**Fill this table:**

| Principle | Evidence | Score (1-5) |
|-----------|----------|-------------|
| Bird-in-Hand | | |
| Affordable Loss | | |
| Crazy Quilt | | |
| Lemonade | | |
| Pilot-in-the-Plane | | |

**Debrief:** Which principle was strongest? Which was weakest? Does a startup need all 5?

### Part C: Effectuation Speed Dating (15 min)
- Stand up. Find someone you haven't talked to yet (90 sec each):
  1. "What's your Bird-in-Hand?"
  2. "What's your Affordable Loss?"
  3. "What's the lemonade opportunity you found?"
  4. "Interested in teaming up?" (if yes → exchange contact)
- Rotate 4-5 times

### Lab Deliverable (submit by end of class)
Your completed "My Effectual Profile" worksheet — submit via LMS or in-class paper.
